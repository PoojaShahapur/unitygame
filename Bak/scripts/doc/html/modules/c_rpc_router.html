Module c_rpc_router

Rpc路由表.

  • 可以设置整个服务的路由，也可以设置某个方法的路由。
  • 可根据服务名和方法名查找目的服ID.
  • 不同玩家有不同的路由表.
  • 全局功能服路由会动态选举目的服。
  • 玩家路由优先于全局功能服路由。
  • 方法名路由优先于服务名路由。

get_dst_svr_id (rpc_clt_id, service, method)
